**Real-Time Patient Flow & Bed Occupancy Analytics Platform**

CareFlow-360 is an end-to-end **real-time and batch data engineering solution** designed for a multi-branch hospital network across India.  
The platform enables **centralized monitoring of patient admissions, discharges, transfers, bed occupancy, and department load**, especially during high-demand periods such as seasonal flu outbreaks.

This project demonstrates a **production-grade Azure data platform** using streaming and batch pipelines built with modern cloud-native services.

## ğŸ§© Business Problem

Hospitals face daily operational challenges due to:
- Lack of real-time visibility into **bed occupancy**
- Inefficient **patient flow management**
- Delayed insights across hospital branches
- No centralized system for operational analytics

CareFlow-360 solves these problems by providing:
- **Near real-time dashboards**
- Historical trend analysis
- Scalable data ingestion from multiple hospitals
